# [515.Find Largest Value in Each Tree Row][title]

## Description
Given the `root` of a binary tree, return an array of the largest value in each row of the tree (**0-indexed**).

**Example 1:**  

![example2](./largest_e1.jpg)

```
Input: root = [1,3,2,5,3,null,9]
Output: [1,3,9]
```

**Example 2:**

```
Input: root = [1,2,3]
Output: [1,3]
```
