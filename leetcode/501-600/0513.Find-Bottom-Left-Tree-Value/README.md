# [513.Find Bottom Left Tree Value][title]

## Description
Given the `root` of a binary tree, return the leftmost value in the last row of the tree.

**Example 1:**  

![example1](./tree1.jpg)

```
Input: root = [2,1,3]
Output: 1
```

**Example 2:**  

![example2](./tree2.jpg)

```
Input: root = [1,2,3,4,null,5,6,null,null,7]
Output: 7
```
